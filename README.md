# A Pizza Date with Sephiroth!
Spend some quality time with a 1st class SOLDIER! Drag a slice of pizza from the whole pizza on the left of the screen and drop it to feed Sephiroth a slice of pizza. Or, you can drag Sephiroth to any seat you'd like!

## ⚠️ NOTE: Intended for desktop applications! ⚠️ Currently may not work on all screen sizes!
